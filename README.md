# 概要

Ruby on RailsとVue.jsの勉強のため作成したポートフォリオです。
Ruby on Railsではユーザ登録機能を実装しています。
また、コントローラでモデルをJSONに出力しています。
Vue.jsでは、JSONでデータを受け取り、ゲーム的な要素でデータに変更を加えています。
ゲーム画面で変更した値がリアクティブにモデルに保存される仕様にしています。

# バージョン
Ruby 2.7.1

Rails 6.0.3
